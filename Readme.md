# Food Hub - Restaurant Website Demo

A modern responsive restaurant website created as a learning project.

## Features

- Responsive Design
- Mobile Navigation Menu
- Digital Menu
- Food Gallery
- Customer Reviews
- Contact Section
- WhatsApp Integration

## Tech Stack

- HTML
- CSS
- JavaScript

## Development Approach

This project was created using AI-assisted development tools and then tested, reviewed, and run locally using VS Code.

## Learning Outcomes

- Website structure
- Responsive design concepts
- Project testing
- Frontend project organization
- GitHub workflow

- Better animations
